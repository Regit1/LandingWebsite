Falstad to Latex Chrome Extension

I was frustrated by how difficult it was to place my circuits from class slideshows into my latex documents, so I made me and my classmates a chrome extension that easily takes the circuit generated by Falstad (export as text) into Latex circuitikz.

Latex Graphing Chrome Extension

I spent so much time trying to generate graphs in latex to look like the function I wanted, so I made a chrome extension that easily takes an input function and bounds, and outputs latex graphing code.
